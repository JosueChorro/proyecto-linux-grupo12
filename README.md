# Proyecto Linux Grupo 6

Este proyecto forma parte de la asignatura *Introducción al Software Libre* de la Universidad de El Salvador.

## Descripción
Implementación de un servidor Linux automatizado utilizando Docker. 
Se integran prácticas de administración de sistemas, control de versiones y virtualización.
